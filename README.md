# OrByZ Studio — Web Oficial

> Construimos experiencias digitales que generan resultados.

Sitio web oficial de **OrByZ Studio**, agencia de desarrollo web y marketing digital especializada en crear sistemas digitales orientados a resultados reales: más leads, mejor posicionamiento y mayor conversión.

🌐 **[orbyzstudio.dev](https://www.orbyzstudio.dev)**

---

## 🚀 Stack Tecnológico

- **Framework:** [Next.js](https://nextjs.org) (App Router)
- **Lenguaje:** TypeScript
- **Estilos:** CSS / Tailwind CSS
- **Fuentes:** [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) con Geist
- **Despliegue:** [Vercel](https://vercel.com)

---

## 📁 Estructura del Proyecto

```
orbyzstudioweb/
├── app/              # Rutas y páginas (App Router)
├── assets/           # Recursos estáticos (imágenes, iconos, etc.)
├── components/       # Componentes reutilizables de UI
├── constants/        # Constantes y configuración global
├── data/             # Datos estáticos (contenido, textos, etc.)
├── features/         # Módulos por funcionalidad
├── hooks/            # Custom React hooks
├── lib/              # Utilidades y helpers
├── public/           # Archivos públicos (favicon, OG images, equipo, etc.)
├── styles/           # Estilos globales
├── next.config.ts    # Configuración de Next.js
├── tsconfig.json     # Configuración de TypeScript
└── postcss.config.mjs
```

## 🛠️ Instalación y desarrollo local

### Prerrequisitos

- Node.js 18+
- npm, yarn, pnpm o bun

### Pasos

```bash
# 1. Clonar el repositorio
git clone https://github.com/orbyz/orbyzstudioweb.git
cd orbyzstudioweb

# 2. Instalar dependencias
npm install

# 3. Iniciar servidor de desarrollo
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver el resultado.
